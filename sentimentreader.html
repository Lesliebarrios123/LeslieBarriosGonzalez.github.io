Sentiment Reader

Python code that returns a files sentiment count

In this project a .txt file is stripped of punctuation and titles. Once the file is broken down it is then compared word for word to two separate files one which is a list of negative words and a second with positive words. This is done with a function that keeps three counts one for each word that is found in one of the lists that we are comparing it to and if a word is not found in one of those two lists it is counted as a neutral word. The function then returns the sentiment of the file.
