Covid-19 Visualization

SQL tables used to visualize in Tableau

In this project SQL functions and queries are used to build tables that would later be used in Tableau to visualize and present the information. The data used for this project was taken from https://ourworldindata.org/covid-deaths from January 2020 to June 2021.

The following skills were used: Joins , CTE's , Temp Tables , Windows Functions , Aggregate Functions , Creating Views , Converting Data Types.
